# Классификатор ТНВЭД (R9)

Автоматическая классификация товаров по ТНВЭД ЕАЭС с верификацией по фото.

## Когда устанавливать

- Компания импортирует товары и заполняет декларации
- Декларант тратит значительное время на ручную классификацию
- Есть выгрузка из 1С с фото, составом, наименованием

## Как работает

1. **Утро (5:00):** сканирует `inbox/classifier/` на новые xlsx-файлы
2. **Подготовка:** извлекает уникальные артикулы + фото из xlsx
3. **Pass 1 (текст):** batch-классификация по текстовым данным (состав, вид ткани, рост, пол)
4. **Pass 2 (фото):** верификация спорных позиций — claude читает фото и проверяет тип изделия
5. **Результат:** xlsx с колонками `AI_ТНВЭД`, `AI_Наименование_ДТ`, `AI_Уверенность`, `AI_Флаги`

## Принцип: фото первично

Фото определяет ТИП изделия (футболка vs блузка vs джемпер).
Текст определяет СУБПОЗИЦИЮ (состав, рост, пол).
Если фото и текст расходятся → флаг `data_conflict`.

## Входной файл

xlsx из 1С с колонками:
- Изображение (встроенные png)
- Артикул, Наименование, Состав (рус), Вид ткани, Размер, Рост, Пол, Страна

## Выходной файл

Тот же xlsx + добавленные колонки:
- `AI_ТНВЭД` — 10-значный код
- `AI_Наименование_ДТ` — таможенное наименование
- `AI_Уверенность` — высокая / средняя / низкая
- `AI_Флаги` — data_conflict, requires_review, requires_clarification
